# 无尽解脱器

## 项目概述
无尽解脱器是一款基于Mumu模拟器的自动化任务管理工具，专为1080×1920分辨率屏幕优化，利用ollama平台的Qwen3-vl大型视觉语言模型API实现智能任务处理。

## 技术规格
- **运行环境**: Mumu安卓模拟器
- **分辨率**: 1080×1920 (竖屏模式)
- **AI引擎**: Ollama Qwen3-vl模型API
- **任务机制**: Interval定时 + Cron计划任务
- **架构模式**: 多开实例，任务独立

## 核心功能

### 1. 多开管理
- 支持同时运行多个Mumu模拟器实例
- 每个实例独立配置，互不干扰
- 实例状态监控和异常恢复(TODO)

### 2. 任务调度系统
#### Interval模式
- 固定时间间隔执行任务
- 可配置间隔时间（秒/分钟/小时）
- 支持执行次数限制

#### Cron模式
- 标准cron表达式调度
- 支持复杂时间规则
- 节假日和特殊日期排除 (TODO)

### 3. 任务类型支持
- **界面操作任务**: 点击、滑动、输入等
- **图像识别任务**: 基于CV2的图像识别
- **文本处理任务**: 基于Qwen3-vl的OCR识别和自然语言处理


## 实现任务示例

### 通用任务
1. **自动登录**
   - 识别登录界面
   - 10分钟等待时间   
2. **日常签到**
   - 仓库收益
   - 探险收益
   - 早晚体力领取
   - 每日奖励(可用，待完善)
   - 忠诚军牌(可用，待完善)
3. **练兵**
   - 目前只支持训练最高等级
   - 不支持训练其他等级
   - 不支持升级等级
4. **火晶实验室**
   - 普通提炼
   - 精炼火晶（每天50%优惠）
5. **银行**
   - 自动存款
   - 自动取款
   - 目前有bug，自动任务时总会失败
6. **地心探险**
   - 自动扫荡
7. **火晶深度探秘**
   - 可用，等完善
8. **竞技场**
   - 自动战斗
   - 选取战力值最小的对手
   - 失败后自动刷新列表
   - 不支持其他检查(英雄、加成等)
9. **宠物寻宝**
    - 自动寻宝
    - 优化高等级宝箱
    - 默认宠物派出
    - 不支持自选宠物
    - 一键收取分享
    - 自动分享宝箱

### 野外任务
1. **统帅领取**
   - 支持领取统帅奖励
   - 支持使用统帅经验（有bug，待完善）
   - 支持开启统帅宝箱（有bug，待完善）
2. **商店**
   - 游荡商人
3. **招募英雄**
   - 自动免费招募
4. **资源收集**
   - 自动采集普矿
   - 自动采集盟矿
5. **自动上车**
   - 按完成进度决定是否上车
   - 支持上车类型(巨龟、收割者、吉娜、佣兵)
   - 不支持队伍列表下拉
   - 不支持强制上车
   - 不支持队列数量限制
6. **自动打怪**
   - 只能打巨龟
   - 30级野兽只在领取体力后自动打，不支持选择
   - 只会派出一支队伍
   - 必须带吉娜
   - 巨兽为队伍7，野兽为队伍8
   - 只支持默认队伍名称
7. **情报处理**
   - 自动处理情报
   - 自动收获情报
8. **更新坐标**
   - 自动更新自己城镇坐标
   - 该功能为自动开罩服务

### 自动打熊
1. **主要功能**
   - 必须开启预约功能
   - 自动开车(间隔350秒)
   - 自动上车
   - 自动开宠物
   - 自动拉回所有队伍
   - 目前不支持英雄换装
2. **使用要求**
   - 车头图标放入指定位置(bear+mumu_id)
   - 开车队伍编为队列1，且必须为默认名称
   - 上车队伍编为队列 2，3，4，5（此处只看位置)

### 联盟任务
1. **联盟捐献**
   - 自动捐献
   - 只支持无限捐献
2. **联盟宝箱**
   - 自动开启宝箱
3. **红包与互助**
   - 自动点击互助
   - 自动领取红包
   - 自动治疗士兵
   - 自动拜访晨曦岛邻居
   - 自动召回驻防中的队伍
   - 城镇被攻击时自动开罩(待测试)

### 其他
1. **晨曦岛**
   - 自动获取晨曦岛收益
2. **更新盟矿**   
   - 自动放置盟矿
   - 需要4阶身份
   - 目前固定在早晚7点40-50
3. **更新上车记录**
   - 更新上车奖励记录
   - 自动上车依据
   - 目前存在bug，巨兽有时莫名为真，待完善
4. **阅读邮件**
   - 自动读取所有邮箱并领取奖励

### 阶段性任务
1. **总动员刷任务**
   - 自动刷新任务
   - 自动领取大小(860,520)拳头任务
   - 中午12点后才会领取小拳头（520）任务   
2. **冰封的宝藏**
   - 自动资历
   
## 配置要求

### 硬件要求
- CPU: 4核以上
- 内存: 8GB以上（每个实例建议2GB）
- 存储: 10GB可用空间

### 软件要求
- Windows 10/11 或 macOS
- Mumu模拟器最新版
- Python 3.8+
- Ollama服务（本地或远程）

### 网络要求
- 稳定互联网连接
- API访问权限
- 低延迟响应（<200ms）
